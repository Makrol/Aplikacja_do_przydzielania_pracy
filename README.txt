Program współpracuje z bazą Oracle 21C

Instrukcja uruchamiania.
1. W bazie danych nalezy uruchomić skrypt table.sql który zawiera wszystkie wymagane tabelki, sekwencje i przykładowe konta użytkowników.
2. Aplikacje można uruchomić poprzez dwukrotne kliknięcie na plik Application.jar
3. Po uruchomieniu należy uzupełnić pola po prawej stronie danymi dotyczącymi bazy w której utworzyliśmy tabelki.

Baza zawiera jedno konto administratora z możliwością dodawania nowych kont pracowników, przydzielania zadań oraz podglądu wszystkich przydzielonych zadań.
login: admin 
hasło: admin

Baza zawiera dwa konta pracowników z możliwością podglądu przydzielonych zadań oraz zgłaszania wykonania zadania.
login: piotr
hasło: piotr

login: adam
hasło: adam
